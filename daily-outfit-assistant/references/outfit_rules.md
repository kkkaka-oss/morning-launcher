# 穿搭规则参考

## 色彩搭配原则

### 安全搭配（适合日常）
- 黑白灰 + 任意彩色
- 同色系深浅搭配
- 米色/驼色 + 咖啡色系
- 藏蓝 + 白色/米色

### 高级搭配
- 莫兰迪色系互搭
- 对比色点缀（小面积）
- 全身不超过3个颜色

## 风格协调原则

### 可混搭
- 优雅 + 慵懒 = 法式风
- 甜美 + 休闲 = 日系风
- 简约 + 通勤 = 职场风

### 避免混搭
- 甜酷 + 优雅复古
- 运动休闲 + 名媛风

## 季节搭配

### 春秋
- 叠穿是关键
- 开衫/薄外套 + 裙装
- 颜色可以明快

### 冬季
- 保暖优先
- 深色打底 + 亮色外套
- 或亮色内搭 + 深色外套

### 夏季
- 轻薄透气
- 浅色为主
- 连衣裙最省事

## 场合建议

### 日常通勤
- 简约款式
- 不夸张的颜色
- 舒适优先

### 约会
- 裙装加分
- 柔和颜色
- 精致细节

### 休闲出行
- 舒适第一
- 方便活动的款式
- 可以更有个性
